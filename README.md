# Программирование мобильных устройств — Практические работы

Вариант 9 | Java | Android SDK 34 | Gradle 8.2

## Структура репозитория

```
├── practical1/
│   ├── basic/      ConstraintLayout с элементами по углам экрана
│   └── advanced/   Конвертер валют (UAH → USD / EUR / GBP / JPY)
├── practical2/
│   ├── basic/      Переход между двумя Activity
│   └── advanced/   Список продуктов с переходом на детали товара
├── practical3/
│   ├── basic/      Передача координат (x, y) между экранами
│   └── advanced/   Фильтр задач по категории (SharedPreferences)
├── practical4/
│   ├── basic/      RecyclerView — список стран и столиц
│   └── advanced/   Страны с поиском, деталями и добавлением
├── Dockerfile
├── docker-compose.yml
└── build-all.sh
```

## Сборка через Docker

### Требования
- Docker
- Docker Compose

### Собрать один проект

```bash
docker-compose run practical1-basic
```

Заменить `practical1-basic` на нужный сервис:
`practical1-basic`, `practical1-advanced`, `practical2-basic`, `practical2-advanced`,
`practical3-basic`, `practical3-advanced`, `practical4-basic`, `practical4-advanced`

### Собрать все проекты

```bash
chmod +x build-all.sh
./build-all.sh
```

APK-файлы появятся в `<название_проекта>/app/build/outputs/apk/debug/`.

## Сборка без Docker (Android Studio / командная строка)

Требования: JDK 17+, Android SDK 34.

```bash
cd practical1/basic
./gradlew assembleDebug
```

## Описание практических работ

### Практическая работа №1
**Тема:** Создание простых приложений в Android Studio

| Блок | Задание (вариант 9) |
|------|----------------------|
| Базовый | ConstraintLayout с элементами, привязанными к углам экрана |
| Усложнённый | Приложение для конвертации валют: ввод суммы, выбор валюты, отображение результата по фиксированному курсу |

### Практическая работа №2
**Тема:** Работа с формами

| Блок | Задание (вариант 9) |
|------|----------------------|
| Базовый | Переход на новый экран (Activity) при нажатии кнопки |
| Усложнённый | Выбор продукта из списка на первом экране, передача данных на второй экран и отображение описания товара |

### Практическая работа №3
**Тема:** Работа с логикой и данными

| Блок | Задание (вариант 9) |
|------|----------------------|
| Базовый | Передать координаты (x, y), введённые на первом экране, и отобразить их в виде формулы: «Координаты: (x, y)» |
| Усложнённый | Фильтр задач: выбор категорий (Работа, Дом, Спорт) на первом экране сохраняется в SharedPreferences; второй экран показывает только задачи выбранных категорий |

### Практическая работа №4
**Тема:** Основы работы со списками (RecyclerView)

| Блок | Задание (вариант 9) |
|------|----------------------|
| Базовый | Список стран и их столиц в RecyclerView (10 стран) |
| Усложнённый | Список стран с поиском, экраном деталей (население, площадь) и экраном добавления новой страны |
